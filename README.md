# appTracker - your easy to use job application tracker

appTracker is the all-in-one job application tracking solution that makes every step of your job search easier.

## **Introduction**

Code Academy alumni need to track their job applications in a central place. This is where appTracker comes in - an easy to use platform where users can create new job applications, move already created applications from one status to another and also delete unsuccessful applications.

## Goals for this project

The main goal of this project was to gain more experience developing a full-stack app while sticking to the AGILE methodology. I also put a little extra emphasis on exploring new technologies to see how well and how fast I would be able to adapt to something new while also consolidating the knowledge of the technologies learned during the academy. 

- Practice developing a full stack application
- Working with new technologies [NIVO](https://nivo.rocks/) for charts and [REACH UI](https://reach.tech/) for modals
- Implementing the application by following User Stories and Wireframes 
- Working AGILE - [Development board](https://github.com/users/lexguru1/projects/1)

## Server-repository

The back-end was built in express and it is a REST API. The database is built using Postgres and Sequelize. You can find the back-end's repository [HERE](https://github.com/lexguru1/appTracker-backend-project)
